Live Status Legend:

  • ↑ (Green Arrow) = Pick is currently winning/covering (On Track)
  • ↓ (Red Arrow) = Pick is currently losing/not covering (At Risk)
  • No Arrow = Pick is pending (game hasn't started) or status is stable

How It Works:

The arrows are dynamically added based on the coverage.diff value:

  • If coverage.diff > 0 → Shows ↑ (Up arrow, green)
  • If coverage.diff < 0 → Shows ↓ (Down arrow, red)
  • If coverage.diff === 0 or no coverage → No arrow

The status badges show:

  • On Track = Currently covering the bet (typically shows ↑)
  • At Risk = Currently not covering (typically shows ↓)
  • Pending = Game hasn't started yet
